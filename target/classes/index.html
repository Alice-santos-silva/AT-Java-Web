Web Cafeteria

Apresentação do Projeto

Bem-vindo à Web Cafeteria, uma aplicação desenvolvida para facilitar a gestão de pedidos e produtos em uma cafeteria. Nosso objetivo é fornecer uma plataforma intuitiva e eficiente para clientes e funcionários, melhorando a experiência de compra e otimizando os processos internos.

Finalidade:

A Cafeteria App foi criada com o propósito de simplificar o processo de pedidos em uma cafeteria, tanto para os clientes quanto para os funcionários. Com uma interface amigável e funcionalidades bem definidas, nosso objetivo é tornar a experiência de compra mais rápida e conveniente.

Objetivos:

    • Permitir que os clientes realizem pedidos de forma rápida e fácil através de uma interface online.
      Facilitar a gestão de produtos e estoque para os funcionários da cafeteria.
      Oferecer uma plataforma intuitiva para acompanhamento e controle dos pedidos em tempo real.
      Melhorar a eficiência operacional da cafeteria, reduzindo erros e aumentando a satisfação do cliente.

    • Funcionalidades Principais:

    • Cadastro de usuários (clientes e funcionários).
      Visualização do cardápio completo de produtos (bebidas, sobremesas, etc.).
      Adição e remoção de itens ao carrinho de compras.
      Gestão de estoque e produtos para funcionários.
